# Strategic Project Analyses

This repository contains consulting-style project reports evaluating feasibility, resource allocation, and critical path management for real-world business projects. The reports demonstrate project analysis, scheduling, and recommendations based on realistic constraints.

## Reports Included

### 1. Next Auckland Scooter Project – Case A
- **Focus:** Children’s scooter product launch  
- **Analysis Includes:**  
  - Baseline schedule feasibility  
  - Critical path activities  
  - Resource allocation and leveling  
  - Cost analysis and cash flow forecasting  
- **Recommendations:** Execution strategy, risk management, and milestone approvals  

### 2. Next Auckland Scooter Project – Case B
- **Focus:** AET Data Centre Migration  
- **Analysis Includes:**  
  - Critical path identification  
  - Resource and dependency risk assessment  
  - Schedule feasibility and sequencing  
- **Recommendations:** Ensuring on-time delivery and minimal downtime  

## Purpose

These reports showcase strategic project management analysis, including:  
- Project feasibility assessment  
- Resource allocation and leveling  
- Critical path identification  
- Cost and cash flow analysis  
- Practical recommendations for stakeholders  

## Usage

- Reports are provided in Word/PDF format.  
- Suitable for demonstrating project management skills, report writing, and analysis methodology.
